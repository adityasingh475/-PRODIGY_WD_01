# 📖 About The Project

This project was developed as **Task-01** during my **Web Development Internship at Prodigy InfoTech**.

The objective was to design and develop a **fully responsive landing page** featuring an interactive navigation menu, smooth scrolling, modern UI, hover effects, and a clean user experience.

---

# ✨ Features

- 🎯 Fully Responsive Design
- 🧭 Fixed Navigation Bar
- ✨ Interactive Hover Effects
- 🚀 Modern Hero Section
- 📜 Smooth Scrolling
- 📱 Mobile Friendly Layout
- 💼 About Me Section
- ⚙️ Services Section
- 📩 Contact Section
- 🔗 GitHub & LinkedIn Integration
- ⬆️ Scroll-to-Top Button
- 🎨 Clean & Modern UI
